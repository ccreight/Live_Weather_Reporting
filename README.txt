A live weather site that pulls from the Google API and OpenWeatherMap API to allow input of city names or latitude and longitude to get information on temperature, humidity, etc.

--------------------------[clarification / instructions] ---------------------------

-Intended resolution is 1920 x 1080
-Homepage is "index.html"
-To add a username straight into the database, you could insert your 
	username then password after line 12 of the SQL script (an account
	with username "admin" and password "root" is provided with one page
	of search history inserted manually if you need that for any reason, too)
-The SQL script is in the project folder as HW3_2.sql - run this to initialize the database
